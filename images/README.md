# Обложки статей

## Как добавить обложку для статьи

### Вариант 1: Автоматическое определение (рекомендуется)

1. Создайте изображение с именем, соответствующим файлу статьи
2. Сохраните его в папку `images/` с расширением `.png` или `.jpg`

**Пример:**
- Статья: `posts/faq-tpr-english-for-kids.md`
- Обложка: `images/faq-tpr-english-for-kids.png`

Система автоматически найдет и использует эту обложку.

### Вариант 2: Явное указание

В файле `index.md` добавьте параметр `image`:

```markdown
{% include article-card.html 
  url="posts/article.md"
  title="..."
  image="/speakid-blog/images/custom-cover.png"
  ...
%}
```

## Требования к обложкам

- **Формат:** PNG или JPG
- **Рекомендуемый размер:** 1200x630px (соотношение 1.91:1)
- **Размер файла:** до 500KB (для быстрой загрузки)
- **Содержание:** должно отражать тему статьи

## Fallback

Если обложка не найдена, автоматически используется `og-default.png` из корня проекта.
